Maven

All versions are located on maven central. If you are using maven the easiest would be to add this to your pom.xml: 
<dependency>
  <groupId>org.jboss.aesh</groupId>
  <artifactId>aesh</artifactId>
  <version>${version.aesh}</version>
</dependency>
For aesh-extensions: 
<dependency>
  <groupId>org.jboss.aesh</groupId>
  <artifactId>aesh-extensions</artifactId>
  <version>${version.aesh}</version>
</dependency>

Using "Gradle":http://www.gradle.org/:

dependencies {
    compile group: 'org.jboss.aesh', name: 'aesh', version: '0.+'
}
