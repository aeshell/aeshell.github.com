Get Involved

There are many ways of getting involved with Æsh and contribute to the project.

We are glad for any involvement either it is finding a bug and report it, contacting us on irc, pinging us on twitter, submit a bug or contribute code on any of our modules.

Chat

Please join us at the #aesh channel on irc.freenode.net to talk about æsh.

Source repository

All the Æsh source code is managed using git and is hosted at github.com. It is all hosted under the Æsh organization. Source repository The different modules:

  • Aesh Main
  • Aesh Extensions
  • Aesh Web Page
