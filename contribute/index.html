Contribute

How to contribute to Aesh

Get Involved

There are many ways of getting involved with Aesh and contribute to the project. We are glad for any involvement either it is finding a bug and report it, contacting us on irc, pinging us on twitter, submit a bug or contribute code on any of our modules.

Chat

We use IRC for communications. Please join us at the #aesh IRC channel on Freenode.

Source repository

All the Aesh source code is managed using git and is hosted at github.com. It is all hosted under the Aesh organization. Source repository The different modules:

  • Aesh Main
  • Aesh Extensions
  • Aesh Web Page

Issue tracker

Aesh use Jira as our tracking system, file issues/requests/etc here: Aesh Jira
